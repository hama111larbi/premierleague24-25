# PremierLeague24-25
An interactive Power BI dashboard project focused on the Premier League 2024-2025, designed to provide a complete overview of both team performance and player efficiency.

The project integrates multiple perspectives:
Team Overview: tracking total matches, goals, assists, points, and rankings with geographical mapping of players’ origins.
Player Overview: analyzing average age per position, assists and goals distribution, and contribution per player.
Keeper & Efficiency: evaluating goalkeepers’ conceded goals, possession efficiency, progression ratios, and disciplinary records (red cards).
This dashboard enables data-driven insights into football performance by combining KPIs, visual analytics, and storytelling in a dynamic and user-friendly environment.

The project highlights how Business Intelligence tools like Power BI can transform raw sports data into actionable insights for decision-making, performance monitoring, and fan engagement.
